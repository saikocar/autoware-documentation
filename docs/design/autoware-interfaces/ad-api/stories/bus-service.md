# バスサービスのユーザーストーリー

## 概要

このユーザーストーリーは、指定された停留所を巡るバスサービスです。

## シナリオ

| ステップ | 操作                                                  | 使用事例                                                                      |
| ---- | ---------------------------------------------------------- | ----------------------------------------------------------------------------- |
| 1    | 自動運転システムを起動します。                     | [起動と終了](../use-cases/launch-terminate.md)                      |
| 2    | 車をガレージから待機位置まで運転します。 | [動作モードの変更](../use-cases/change-operation-mode.md)            |
| 3    | 自律制御を有効にします。                                 | [動作モードの変更](../use-cases/change-operation-mode.md)            |
| 4    | 次のバス停まで車を運転します。                    | [指定された場所までの運転](../use-cases/drive-designated-position.md) |
| 5    | 車両に乗り降りします。                                | [乗降](../use-cases/get-on-off.md)                              |
| 6    | 終点のバス停でない場合は、ステップ4に戻ります。            |                                                                               |
| 7    | 車両を待機位置まで運転します。                 | [指定された場所までの運転](../use-cases/drive-designated-position.md) |
| 8    | 車両を待機位置からガレージまで運転します。 | [動作モードの変更](../use-cases/change-operation-mode.md)            |
| 9    | 自動運転システムを停止します。                    | [起動と終了](../use-cases/launch-terminate.md)                      |
